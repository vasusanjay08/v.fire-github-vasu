Let us discuss what is meant by algebraic identities and what is the use of that identities?


In mathematics,an identity elment, or neutral element, of a binary operation operating on a set is an element of set which leaves unchanged every element of the set when the operation is applied. This concept is used in algebraic structures such as groups and rings. Theterm identity element is often shortened to identity (as in the case of additive identity and multiplicative identity) when there is no possibilty of confusion, but operation it is assoiated with.
